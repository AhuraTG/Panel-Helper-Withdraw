# Panel Helper - Withdraw

این افزونه جهت افزایش سرعت و دقت در بخش واریز و برداشت طراحی شده است.

امکانات
* ساخت خودکار فرمت بررسی تراکنش
* کپی سریع اطلاعات واریز
* انتخاب خودکار پنل مورد نظر ( جت . آس )
* ثبت و بروزرسانی قیمت ارزهای واریزی کاربران جهت ارجاع به بخش مالی
* محاسبه مبلغ ریالی واریزی کاربران
* استعلام سریع واریزهای ارزی
* پنل اختصاصی مدیریت واریز و برداشت
* بهینه سازی فرآیند بررسی تراکنش ها

نسخه فعلی: 1.1

توسعه دهنده ( اهورا )
 این پروژه صرفاً جهت استفاده پشتیبانی ( بخش تیکت ) مجموعه در نظر گرفته شده و دسترسی به سورس آن محدود می باشد.
